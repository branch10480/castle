# テーマ一覧

`nodeStyle` と `edgeStyle` はここから選ぶ。必ずユーザーに選択させる。  
迷っている場合は、SKILL.md の最小テンプレートで各テーマのローカル `.drawio` をプレビュー生成する。

## crisp-mono（シャープ・モノクロ）
- 見た目: 余計な装飾なし、コントラスト強め、直線的。技術資料向き。
- nodeStyle:
  rounded=0;whiteSpace=wrap;html=1;fillColor=#FFFFFF;strokeColor=#111111;fontColor=#111111;strokeWidth=1;
- edgeStyle:
  edgeStyle=orthogonalEdgeStyle;rounded=0;endArrow=block;strokeColor=#111111;strokeWidth=1;
