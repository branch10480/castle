# イオンモール座間 駐車場空き状況確認

イオンモール座間の駐車場のリアルタイム空き状況を確認してください。

## 手順

1. Chrome拡張機能を使用して https://zama-aeonmall.com/static/detail/access にアクセス
2. ページのスクリーンショットを撮影
3. 左側サイドバーの「駐車場ご利用状況」から各階の状況を読み取る
4. 以下の形式で結果を報告

## 出力形式

```

### 立体駐車場
| 階 | 状況 |
|----|------|
| 1F | 🔴満 / 🟡混 / 🔵空 |
| M2F | ... |
| 2F | ... |
| 3F | ... |
| 4F | ... |
| RF | ... |

### その他の駐車場
| エリア | 状況 |
|--------|------|
| 本棟屋上駐車場 | ... |
| A平面駐車場 | ... |

### おすすめ
空いている階を案内
```

## 基本情報
- 住所: 神奈川県座間市広野台2丁目10-4
- 収容: 約2,500台
- 料金: 全日無料
- 時間: 7:00〜24:00

## ステータスの意味
- 🔵空（青）: 空きあり
- 🟡混（黄）: やや混雑
- 🔴満（赤）: 満車
