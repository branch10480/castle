# イオンモール座間 駐車場空き状況確認

## Description
イオンモール座間の駐車場のリアルタイム空き状況を確認するスキル。各階（1F, M2F, 2F, 3F, 4F, RF）、本棟屋上駐車場、A平面駐車場の満空情報を取得して表示する。

## Trigger
- 「座間イオン 駐車場」
- 「イオンモール座間 駐車場」
- 「ザマイオン 駐車場」
- 「座間モール 空き」

## Instructions

### 1. ブラウザでアクセスページを開く
公式の駐車場空き状況ページにアクセスする：
- **URL**: https://zama-aeonmall.com/static/detail/access

### 2. スクリーンショットを取得
ページ左側サイドバーに「駐車場ご利用状況」が表示されているので、スクリーンショットを撮影して空き状況を確認する。

### 3. 結果を整理して報告
以下の形式で空き状況を報告する：

```
## 🅿️ イオンモール座間 駐車場リアルタイム空き状況

### 立体駐車場
| 階 | 状況 |
|----|------|
| 1F | 🔴満 / 🟡混 / 🔵空 |
| M2F | ... |
| 2F | ... |
| 3F | ... |
| 4F | ... |
| RF | ... |

### その他の駐車場
| エリア | 状況 |
|--------|------|
| 本棟屋上駐車場 | ... |
| A平面駐車場 | ... |

### おすすめ
空いている階を案内する
```

## 駐車場基本情報
- **住所**: 〒252-0012 神奈川県座間市広野台2丁目10-4
- **収容台数**: 約2,500台
- **料金**: 全日無料
- **利用時間**: 7:00〜24:00

## 混雑傾向
- 土日祝の11時〜15時が最も混雑
- 午前中（開店直後）や夕方以降がスムーズ
- 上層階（4F, RF）が比較的空いている傾向

## 状況の見方
- 🔵 **空**（青）: 空きあり
- 🟡 **混**（黄）: やや混雑
- 🔴 **満**（赤）: 満車
